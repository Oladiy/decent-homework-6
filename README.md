# **Project description**

Программа при запуске получает и выводит данные о валидаторах публичной сети Polkadot.

Список данных:
1. Адрес валидатора
2. Информация о балансе (включая зарезервированные и замороженные средства)
3. Комиссия
4. Ledger

# **Install**

`npm install`

# **Run**

`nodejs src/getValidatorsInfo.js`
